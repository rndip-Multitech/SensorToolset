# ./config

To use a configuration file with the application, include this directory along with a sample configuration file and documentation in the packaged mPower custom application.

## Configuration File: radioBridgeTools.cfg.json

The configuration file contains the following settings:

- `host`: The host address to bind the server to (default: "0.0.0.0")
- `port`: The port number to run the server on (default: 5000)
- `debug`: Enable debug mode for Flask (default: false)
