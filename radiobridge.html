General Sensor Configuration
Use these settings to configure a RadioBridge Sensor's general configuration parameters, including Supervisory Period, ADR, Confirmed Messages, etc.

The hexcode generated can be used with any RadioBridge sensor.
Advanced Sensor Configuration
Use these settings to configure any RadioBridge Sensor's advanced configuration parameters.

The only adjustable setting is Link Quality Message. This setting determines how often the sensor sends a message that contains RSSI and SNR signal statistics as received by the sensor.
