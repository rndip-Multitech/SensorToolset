Update RadioBridge Decoder

Fetch the latest RadioBridge decoder bundle into this gateway. This keeps RadioBridge devices in sync with their official decoding logic while preserving local app integrations.

Configure the source URL in config/radioBridgeTools.cfg.json (radiobridge_decoder_url) or set the RADIOBRIDGE_DECODER_URL environment variable. A 502 Bad Gateway means the server could not reach that URL (wrong URL, network, or upstream error).
