How to configure alerts
  1. Set up email under Notifications → Configure delivery (recipient and SMTP). Alerts are sent from the Sensor Monitor page while it is open and receiving uplinks.
  2. Choose a sensor so the toolkit can list decoded fields from recent traffic.
  3. State change — pick one field; an email is sent when that field’s value differs from the previous uplink for this sensor (the first packet after you open the page establishes the baseline).
  4. Threshold + duration — pick a field, how it should compare to a number, a threshold, and how long the condition must stay true before one email is sent.
