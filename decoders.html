Add / Edit Decoder (JavaScript)

Upload a decoder file

Upload a `.js` file that calls `window.RBTDecoders.registerDecoder(...)`.

Or paste decoder code

If you paste a TTN/ChirpStack decoder, click Wrap / Fix format to convert it into a Sensor Toolkit plugin automatically. “Test-load” runs the code in this page only (via a Blob). “Save” uploads it to the server so `uplinks.html` can load it.

Test decoder with payload (debug)

Enter a hex or base64 payload and run a decoder to see the decoded output. Useful for debugging decoders without live uplinks.

Enter one or both; hex is used if both are set. Spaces and 0x prefixes in hex are ignored.
